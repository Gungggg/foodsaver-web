# DESIGN.md

# FoodSaver Design Documentation

## Overview

Dokumen ini mendefinisikan standar desain UI/UX untuk aplikasi FoodSaver. Seluruh halaman harus mengikuti prinsip:

* Clean Interface
* Sustainability Theme
* Modern Marketplace
* Accessibility First
* Mobile First
* Consistent Design System

---

# Design Principles

## 1. Simplicity

Interface harus sederhana dan mudah digunakan oleh:

* Mahasiswa
* Pekerja muda
* Merchant UMKM
* Admin Platform

---

## 2. Sustainability

Visual harus mencerminkan:

* Ramah lingkungan
* Pengurangan limbah makanan
* Green Technology
* Circular Economy

---

## 3. Trust & Safety

Visual harus memberikan kesan:

* Aman
* Profesional
* Terpercaya

---
